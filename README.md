# 💰 Smart Finance App

Personal financial management mobile application with artificial intelligence.

## 🚀 Features

-  **Dashboard** - Overview of monthly expenses
-  **AI Predictions** - Forecast of next month’s spending
-  **Smart Alerts** - Detection of anomalies and overspending
-  **Automatic Categorization** - Machine learning to classify transactions
-  **Analytics & Trends** - Visualization of spending patterns

## 🛠️ Technologies

### Frontend (Mobile)
- React Native / Expo
- React Navigation
- JavaScript

### Backend (API)
- Python 3.11+
- FastAPI
- SQLAlchemy
- Scikit-learn
- SQLite

## 📦 Installation

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Mobile App
```bash
cd mobile-app
npm install
npm start
```
